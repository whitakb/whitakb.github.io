My HTML/CSS Cheat Sheet

File Tags (Document Structure)

Tag Name Description
<!DOCTYPE> Document Type Tells what version of HTML the document is written in
<html> HTML tag Top-level element of the HTML file (include language "lang" for accessability)
<head> Document Header Holds meta data for the document
<body> Body Wrapper for the main contents of the document

Header Tags (in <head>)

Tag Name Description
<title> Document Title Shows in the browser tab or title bar
<meta> Meta tags See notes below on meta tags
<script> Script Defines a section for embedded client-side scripts (JavaScript)
<noscript> No Script Define alternate content for sessions that have disabled scripts
<style> Style Defines a section for internal CSS definitions
<link> Link Defines a link to an external style sheet
<base> Base Specifies a default URL and target for all links on the page

Meta Tags (in <head>)

Attribute/Name Example Description
charset <meta charset="UTF-8"> Character encoding. UTF-8 is the standard
viewport <meta name="viewport" content="width=device-width, initial-scale=1.0"> Renders the page to the width of the device's screen & controls the zoom level
X-UA-Compatible <meta http-equiv="X-UA-Compatible" content="ie=edge"> Specifies the document mode for Internet Explorer. IE=Edge is the highest supported mode.

Semantic Tags (in <body>)

Tag Name Description
<header> Header For introductory content or a set of navigational links
<nav> Navigation For a set of navigational links
<section> Section A section is a thematic grouping of content, typically with a heading
<article> Article independent, self-contained content, like a blog post or user comments or newspaper article
<footer> Footer Defines a footer with, for example, copyright, site map, back to top, etc.
<aside> Aside Defines some content aside from the content it is placed in (like a sidebar)
<details> Details Defines additional details that the user can view or hide
<figure> Figure Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
<figcaption> Figure Caption defines a caption for a <figure> element. The <figcaption> element can be placed as the first or as the last child of a <figure> element
<img> Image Defines the image for a <figure> tag
<main> Main Specifies the main content of a document
<mark> Mark Defines marked/highlighted text
<summary> Summary (for details) Defines a visible heading for a <details> element
<time> Time Defines a date/time
You will find HTML pages with <section> elements containing <article> elements, and <article> elements containing <section> elements.

Hyperlink Attributes (<a>)

Type Option Description Example Try it
attribute href Indicates the link's destination <a href="http://www.google.com">Google</a> Google
attribute target Indicates where the target page should be opened <a href="http://www.google.com" target="_blank">Google</a> Google
attribute value target="_blank" Opens in new tab <a href="http://www.google.com" target="_blank">Google</a> Google
attribute value target="_self" Opens in same frame as where clicked (default) <a href="http://www.google.com" target="_self">Google</a> Google
attribute value target="_parent" Opens in parent frame <a href="http://www.google.com" target="_parent">Google</a> Google
attribute value target="_top" Opens in full body of the window <a href="http://www.google.com" target="_top">Google</a> Google
attribute value target="framename" Opens in the named iframe
